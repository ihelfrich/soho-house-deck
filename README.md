# Soho House: forensic findings deck

A 17-slide HTML presentation summarizing an independent forensic investigation
into Soho House & Co. Self-contained single file (`index.html`); D3 and Google
Fonts load from CDN.

Every sharp claim is evidence-tiered: T1 verified against a primary source,
T2 strong and multiply-sourced, T3 single-source lead, T4 analyst interpretation.
The full investigation corpus, sources, and verification log live separately.

## Viewing

Open `index.html` in any modern browser, or use the live site.
Navigate with arrow keys, space, or by clicking the left/right half of the page.

## Hosting

Private repository with a GitHub Pages site. The published site is reachable by
anyone with the URL but is not linked or indexed.
